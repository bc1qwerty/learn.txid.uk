The **Regression Theorem** is a theory presented by Mises that demonstrates how the value of all money, traced back through past exchange rates, originally derives from non-monetary uses (value as a commodity).

The logical flow:

1. The reason we accept money today is because someone else accepted it yesterday
2. The reason they accepted it yesterday is because someone accepted it the day before
3. Tracing back this way, we eventually reach a point where the substance had value not as money, but as a commodity
4. Gold originated from non-monetary value in jewelry and industrial uses, and eventually became money

Bitcoin and the Regression Theorem:

- Critics: Bitcoin has no non-monetary use, so it cannot become money
- Counterargument: Bitcoin's initial value originated from **services**—censorship-resistant P2P transfer. The Regression Theorem does not require only physical commodities
- In reality, Bitcoin underwent this process between 2009-2010, forming prices starting from zero
