**Node** is software that stores a complete copy of the Bitcoin blockchain and independently verifies the validity of all transactions and blocks.

By running a full node:
- You verify all rules yourself (21M cap, consensus rules, etc.)
- You can check balances and transactions without trusting anyone
- It's like voting to enforce the network's consensus rules

Why nodes matter:
- **"Don't trust, verify"**: Trusting third parties is a security vulnerability
- **The reality of decentralization**: No matter how powerful miners are, nodes enforce the rules
- **Censorship resistance**: You can broadcast transactions directly through your own node

You can run a full node on low-spec computers like a Raspberry Pi. This low barrier to entry sustains Bitcoin's decentralization.
